# Searching

## Search
The search is mostly done with the help of the tags. But there are other search operators that can be used.

### Search operators
#### ``album``
Example: ``album: Some album with nice pictures``
Only show images from a specific album this does not uses contains so the album name needs to be exact.

#### ``order_by``
Example: ``order_by: exif_shot_date_time``
Order the results by a specific fielt see the list of fields somewhere below (@todo).

#### ``orientation``
Example: ``orientation: landscape``
Options: ``vertical``, ``horizontal``, ``square``
